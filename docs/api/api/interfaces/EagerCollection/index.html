mapReduce()​

mapReduce<K2, V2, MapperParams>(mapper, ...mapperParams): <Accum, ReducerParams>(reducer, ...reducerParams) => EagerCollection<K2, Accum>

Create a new eager reactive collection by mapping some computation mapper over this one and then reducing the results with reducer. Note that this function is curried so that mapper and reducer rest parameters can be provided separately, e.g. as mapReduce(MyMapper, foo, bar)(MyReducer, baz)
